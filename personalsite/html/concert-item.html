Project: Concert

Group Study/Tutoring

This project was done with a group of fellow seniors as a senior design project. The purpose of this project was to help students overcome social disconnections, especially with recent Covid lockdowns, in their academics. The website is a tool for students and tutors to be able to interact, message, and form groups to study. This website utilitizes multiple API's such as Calendly to help students schedule appointments.

My role in this project was to take charge of the student UI, FAQ subpage, and the backend along with these components. With this, I was also tasked with creating the messaging system in the web application for students to communicate. I was also one of the members tasked with testing the components including: UI manual testing, usability testing, and accessibility testing. This was my first major involvment in a web application that I have worked on.
