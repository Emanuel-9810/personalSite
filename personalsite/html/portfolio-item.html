Project: Monkey Fitness

Health App

This android application was worked on with a group of peers for a software engineering project. Our group created a health/fitness application which gives its users the ability to keep track of and set their fitness goals, see their progress, track caloric history, fitness history, and see current news on health and fitness. This application was not released on Google playstore, however application's code is up to download on my github.

My role in this project was to create the homescreen page as well as help out other members with their pages when needed. As this was all of our firt time's creation of a mobile application, we all pitched in to each other's work to make sure we were all up to speed and maintaining our waterfall model. This project was done with Java using Android Studio.
